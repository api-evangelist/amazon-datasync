# Amazon DataSync API

AWS DataSync is an online data transfer service that simplifies, automates, and accelerates moving data between on-premises storage systems, AWS storage services, and other cloud storage. DataSync can transfer data at speeds up to 10 times faster than open-source tools.

## APIs

- **Amazon DataSync REST API** - RESTful API for AWS DataSync enabling management of data transfer tasks, locations, agents, and task executions for automated data movement between storage systems.
