S3 server access logs can now be delivered directly to Amazon CloudWatch Logs, where they are parsed into structured fields and ready to query within seconds. The capability supports CloudWatch Logs Insights for on-demand analysis, metric filters for proactive monitoring, and optional S3 Tables integration for SQL analytics via Apache Iceberg format, removing the need for custom ETL pipelines.
