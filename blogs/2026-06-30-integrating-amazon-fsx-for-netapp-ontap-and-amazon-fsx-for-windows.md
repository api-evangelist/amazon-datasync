This post shows how to connect FSx for NetApp ONTAP and FSx for Windows File Server to Microsoft Entra ID on Azure using Entra Domain Services. The solution bridges AWS storage services with Azure-based identity management through managed domain controllers that synchronize users and credentials, exposing traditional Active Directory protocols over a VPN tunnel for SMB authentication.
